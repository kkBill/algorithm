### 关于

本仓库用于基础算法练习，主要包括LeetCode，以及基于不同语言对常见数据结构的练习实现。

当学习一门新语言时，为了熟练其语法，常常用这种语言来写几道题，实现一些相对比较复杂的数据结构。
